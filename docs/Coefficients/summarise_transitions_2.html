Tobacco

Tobacco consumption is measured in the usual number of cigarettes smoked per day, and is taken directly from a US variable (ncigs).

We estimated a zero inflated poisson model, with the formula:

ncigs ~ age + sex + SF_12 + labour_state + job_sec + ethnicity + hh_income ethnicity + labour_state + SF_12
